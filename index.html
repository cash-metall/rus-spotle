Как играть

Угадайте загаданного артиста за 8 попыток
1

Введите в поиске имя любого артиста из базы и выберите его из списка.

2

Откроются 6 плиток с атрибутами вашего артиста, окрашенных по «похожести» на артиста дня.

3

Используйте подсказки, чтобы за 8 попыток вычислить и угадать точное имя.

Точное совпадение Близко Не совпадает

Стрелки-подсказки указывают на артиста дня: «Дебют» — ▲ позже (год больше), ▼ раньше; «Популярность» — ▲ популярнее, ▼ менее популярен.

🗓 Дебют

Год первого релиза (альбом / мини-альбом / микстейп). Зелёный — точно, жёлтый — в пределах 5 лет, со стрелкой направления.

👥 Состав

Число участников в самом известном составе. «Соло» — для сольных артистов. Только точно / не совпало.

🔥 Популярность

Место в рейтинге популярности базы (№1 — самый популярный). Жёлтый — рядом по рейтингу. Стрелка: ▲ — артист дня популярнее вас, ▼ — менее популярен.

🚻 Пол

Мужской / Женский / Смешанный / Небинарный. Только точно / не совпало.

🎸 Жанр

Один основной жанр. Только точно / не совпало.

📍 Город

Город происхождения. Зелёный — тот же город, жёлтый — тот же регион/страна.
